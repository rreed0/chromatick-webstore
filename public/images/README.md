# Product images

Add your product photos here. Update the paths in `lib/products.ts` when you add real images.

**Structure:**
- `placeholder.svg` – Fallback used until you add real photos
- `products/{slug}/` – e.g. `lf-20w-green/`, `f91w-green/`
  - `thumb.jpg` – Thumbnail for the main shop page
  - `1.jpg`, `2.jpg`, `3.jpg` – Gallery images for the product page
